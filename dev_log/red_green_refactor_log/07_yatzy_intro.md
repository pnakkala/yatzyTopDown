# The end goal: Console-based Yahtzee game

## [Yahtzee](https://en.wikipedia.org/wiki/Yahtzee)
- player rolls 5 dice
- player can re-roll up to 3 times
- player chooses a scoring 'category'
- game ends when all scoring categories are chosen
